# Two Pointers

## When To Use

정렬된 배열, 연속 구간, 양끝에서 좁혀야 하는 탐색에서 사용합니다.

## Core Idea

두 인덱스를 움직이며 전체 경우를 중복 없이 줄입니다. 포인터 이동 규칙이 단조성을 가져야 시간복잡도를 줄일 수 있습니다.

## Complexity

- Time: 보통 `O(n)` 또는 정렬 포함 `O(n log n)`
- Space: 보통 `O(1)`

## Pitfalls

- 포인터 이동 조건이 애매하면 무한 루프가 생깁니다.
- 정렬이 필요한 문제라면 원래 인덱스를 잃어도 되는지 먼저 확인합니다.
- 같은 값 중복 처리 조건을 명확히 정합니다.
